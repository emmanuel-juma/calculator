# 🧮 Calculator

A simple, responsive calculator built using **HTML**, **CSS**, and **JavaScript**. It performs basic arithmetic operations through a clean and user-friendly interface.

## ✨ Features

* Addition, subtraction, multiplication, and division
* Responsive design for desktop and mobile devices
* Clear (AC) button
* Delete (DEL) button
* Decimal number support
* Real-time display updates

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/emmanuel-juma/calculator.git
```

2. Navigate to the project folder:

```bash
cd calculator
```

3. Open `index.html` in your browser.

No additional dependencies or installation are required.
